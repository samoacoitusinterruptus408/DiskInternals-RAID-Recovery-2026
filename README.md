## 💡 Troubleshooting common issues

Users sometimes encounter challenges during the recovery process. Review these points if you face issues:

*   The software does not show my drives: Ensure all hardware connections are secure. Check if your computer detects the drives in Disk Management. If Windows does not see the physical drives, the software cannot see them.
*   The recovery hangs: Close other background programs that use your hard drives. Close your antivirus software during the scan to prevent interference.
*   The scan shows incorrect data: Verify the RAID array settings. If you chose the wrong RAID type or block size, the scan findings will show as damaged or unrecognizable. Run the scanner again with different parameters.
*   The system runs slow: Data recovery consumes high amounts of CPU and disk bandwidth. Expect your computer to feel sluggish while the scan runs. Allow the process to complete in the background.
